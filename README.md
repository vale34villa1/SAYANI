# Protegidas
 Protegidas es una plataforma integral diseñada para erradicar la explotación infantil y prevenir la deserción escolar en niñas y adolescentes entre 3 a 21 años en Perú. A diferencia de las soluciones actuales, combina monitoreo biométrico, IA preventiva y tecnología offline-first para proteger a las menores en zonas rurales y urbanas.
